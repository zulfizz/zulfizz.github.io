Cara Membuat Github Pages

Posted 20. April 2019. 1 min read.

Github Pages adalah sebuah layanan hosting web statis yang disediakan oleh Github. Ketika mendaftar di Github otomatis bisa menggunakan layanan ini dan akan mendapatkan subdomain .github.io. Saya sendiri sudah mencobanya dan proses pembuatannya pun tidak terlalu rumit seperti hosting pada umumnya. Kita hanya perlu sedikit pengetahuan tentang GIT. Contoh halaman yang saya buat bisa dilihat di zulfizz.github.io.

Sebelum membuatnya, pastikan sudah mendaftar di Github dan agar nantinya lebih mudah, install juga Git di lokal. Setelah semuanya siap berikut langkah-langkahnya.

1. Buat Repository Baru di Github

Langkah pertama adalah membuat repository baru di Github dengan menggunakan username Github dan diikuti subdomain .github.io. Contoh zulfizz.github.io.

2. Clone Repository

Sebenarnya pada langkah pertama anda sudah berhasil membuat halaman Gitub. Akan tetapi masih belum ada isinya maka dari itu kita akan mengisi repository yang sudah dibuat dengan file index.html agar ketika mengakses halaman yang sudah dibuat tadi bisa muncul tampilanya. Buka terminal atau CMD dan pastikan sudah menginstall git. lalu ketikkan perintah berikut.

git clone https://github.com/zulfiizz/zulfizz.github.io.git github-pages
cd github-pages
echo "Hello World! Selamat datang" >> index.html
git init
git add index.html
git commit -m "add index.html"

3. Push ke Repository Github

Setelah mengisi repository lokal langkah selanjutnya adalah mensinkronkannya dengan remote repository fi Github.

git push -u origin master

4. Lihat Hasilnya

Apabila kita mengikuti langkah-langkah diatas dengan benar maka seharusnya halaman Github kita sudah bisa diakses. Contohnya zulfizz.github.io.
