Apa Itu Markdown?

Posted 14. April 2019. 2 min read.

Markdown adalah sebuah alat untuk mengonversi text-to-HTML yang biasanya digunakan untuk membuat konten pada sebuah web. Cara kerja markdown yaitu mengubah format penulisan berupa plain text kedalam bentuk HTML pada web. Bekerja dengan markdown membuat kita lebih mudah dalam membuat format HTML karena konsep markdown sendiri yaitu easy-to-read dan easy-to-write.

Markdown pertama kali dibuat oleh John Gruber pada tahun 2004 dengan menggunkan Perl sebagai bahasanya. Markdown sendiri dibuat bertujuan untuk mempermudah pembuatan konten HTML maupun XHTML dalam bentuk plain text.

Sudah banyak situs web besar dan terkenal yang menggunakan markdown untuk membuat kontennya. Sebut saja Github, Reddit, Stackoverflow, SourceForge, dan yang lainnya. Artikel ini sendiri juga ditulis dengan menggunakan markdown.

Seperti tipe file yang lainnya, markdown memiliki beragam jenis extensi file. Antara lain .markdown, .mdown, .mkdn, .md, .mkd, .mdwn, .mdtxt, .mdtext, .text. Beberapa situs menerapkan standarisasi extensi untuk file markdown yang berbeda.

Contoh Penulisan Markdown

# Markdown
**Markdown** is a text-to-HTML conversion tool for web writers. 
Markdown allows you to write using *easy-to-read*, *easy-to-write* plain text format, then convert it to structurally valid XHTML (or HTML).

[Markdown](http://daringfireball.net/projects/markdown/ "Markdown")

"Markdown" is two things:

1. Plain text formatting syntax.
2. Software tool.
