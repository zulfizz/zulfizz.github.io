Cara Membuat Jaringan AdHoc di Windows

Posted 1. November 2016. 1 min read.

Jaringan AdHoc merupakan salah satu jenis WLAN yang terdiri dari sekumpulan node-node yang saling terhubung satu sama lain tanpa melibatkan perangkat tambahan seperti accsess point. Untuk membuat adhoc di windows kita perlu menggunakan bantuan CMD untuk mengatur SSID dan Password dari jaringan yang kita buat.

  1. Periksa driver
    Untuk mengecek apakah windows yang dipakai support untuk adhoc atau tidak maka buka CMD (Run as Administrator) lalu ketikkan perintah berikut

    netsh wlan show drivers

    Lalu tekan enter, dan lihat pada Hosted Network Support, apabila tertulis Yes berarti perangkat yang kita pakai support untuk jaringan ini.

  2. Setting SSID & Password
    Ketikkan perintahh berikut untuk mengatur SSID dan password jaringan adhoc yang akan dibuat

    netsh wlan set hostednetwork mode=allow ssid=NAMA_JARINGAN key=PASSWORD

    Ganti ssid dan password sesuai keinginan dan untuk password minimal berjumlah 8 karakter.

  3. Aktifkan Jaringan Adhoc
    Setelah mengatur jaringan adhoc, langkah selanjutnya adalah mengaktifkannya dengan cara mengetikan perintah berikut di CMD

    netsh wlan start hostednetwork

    lalu tekan enter, makan otomatis AdHoc akan berjalan dan pastikan wifi di laptop sudah berjalan.

  4. Sharing koneksi (Opsional)
    Langkah terakhir adalah menyambungkan jaringan adhoc yang sudah kita buat dengan internet. Caranya yaitu masuk ke "Network Sharing Center" melalui control panel. Selanjutnya pada bagian kiri pilih "Change Adapter Setting" dan klik kana pada "Wifi" atau adapter yang tersambung ke internet lalu pilih "Properties". Pilih tab "Sharing" kemudian centang pada "Allow other network uers to connect through this computers internet connection" dan pada pilihan dibawahnya pilih "Local Area Connection2", lalu apabila muncul peringatan klik Ok.

Itulah sedikit cara untuk membuat jaringan AdHoc di Windows 8 atau 8.1. Untuk menghentikan AdHoc kita cukup jalankan CMD sebagai Administrator dan ketikkan

netsh wlan stop hostednetwork

lalu tekan enter maka otomatis jaringan akan berhenti.
