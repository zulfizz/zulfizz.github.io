Perjalanan Menjadi Programmer - Part 0

Posted 19. June 2020. 2 min read.

Ini adalah cerita perjalanan saya dari awal sampai bisa berprofesi sebagai Web Developer. Dari yang awalnya penasaran cara kerja web di internet sampai akhirnya kenal dengan platform blogger untuk pertama kalinya pada saat kelas 2 SMP. Namun saat itu rasa penasaran saya belum terbayarkan karena belum benar-benar tau bagaimana caranya sebuah web bekerja. Karena saat itu yang paling mudah untuk dipelajari adalah blogger, maka saya mulai membuat blog untuk pertama kalinya, tak lupa juga untuk ikut bergabung dengan forum-forum blogger di facebook. Menulis artikel pun masih melihat punya orang lain sebagai referensi atau melihat buku catatan sekolah.

Setelah berhasil membuat blog di blogger, rasa penasaran saya kembali muncul saat ingin tahu bagaimana tulisan yang saya buat bisas muncul di blog dengan rapi. Dari situ mulailah untuk mencari cara memodifikai tampilan dari blogger. Bermodalkan google untuk browsing kesana-kesini saya jadi tahu ternyata blogger menggunakan template untuk menampilkan kontennya. Akhirnya saya mencoba mencari template-template gratis untuk diterapkan di blog saat itu. Selain blogger, saya juga mencoba menggunakan CMS lainnya seperti Wordpress, Joomla, MyWapBlog yang sekarang sudah tutup, padahal banyak kenangannya :(. Saya mencoba semua itu tak lain hanya karena rasa penasaran dan saat itulah mulai tertarik untuk melanjutkan ke jurusan IT setelah lulus SMP. Oiya, saat itu saya belum memiliki laptop maupun komputer sendiri, jadi masih pinjam netbook lenovo kecil milik kakak yang digunakan untuk kuliah.

Cerita berlanjut saat pertama kali masuk SMK pada tahun 2016. Sebelumnya, saya memutuskan untuk masuk ke SMK jurusan TKJ (Teknik Komputer dan Jaringan) karena memang jurusan tersebut sudah diidam-idamkan sejak SMP. Setelah kelulusan tiba, saya langsung mencari dan membandingkan beberapa SMK yang ada jurusan TKJ dari mulai yang negeri sampai swasta. Singkatnya saya berhasil diterima di SMK 1 Kedungwuni dengan hasil yang alhamdulillah cukup memuaskan.

Di awal-awal sekolah, saya tidak pernah mengira jika akan ada mata pelajaran pemrograman karena setau saya saat itu di jurusan TKJ yang akan dipelalari tidak jauh-jauh dari jaringan internet dan server. Ternyata pada saat itu ada pelajaran pemrograman dasar dan pemrograman web sebagai pelajaran tambahannya. Di mata pelajaran Pemrograman Dasar saya mendapat materi awal yaitu dasar dasar algoritma dan pemrograman dengan Pascal. Sedangkan di mata pelajaran Pemrograman Web pada saat itu materi yang diberikan adalah html, css, dan sedikit javascript. Sebenarnya ada materi untuk PHP dan MySql namun karena keterbatasan waktu akhirnya di akhir semester 2 tidak sampai ke materi tersebut.

Di mapel Pemrograman Web saat semester pertama, semua siswa diwajibkan untuk membuat blog dan mengisi dengan artikel sebagai nilai tambahan. Saya yang saat itu sudah pernah menggunakan blogger tidak terlalu kesulitan untuk tugas, hanya tinggal mengaktifkan kembali blog saya yang sudah lama tidak terurus. Bahkan ada beberapa teman yang meminta bantuan ke saya karena kesulitan membuat blog.

Setelah beberapa minggu pertemuan di jam pemrograman, saya mulai tertarik untuk mempelajarinya lebih lanjut. Saya mulai mencari tutorial-tutorial html css di internet, namun kesulitan saya saat itu adalah tidak bisa cepat memahami sesuatu yang baru. Untungnya saat itu saya punya teman yang sama-sama tertarik di bidang pemrograman sehingga kami sering bertukar pertanyaan dan belajar bareng.
