Pengenalan Dasar HTML 5

Posted 11. September 2016. 2 min read.

Sebelum berkenalan dengan HTML 5 terlebih dahulu kita cari tahu apa itu HTML. HTML merupakan singkatan dari Hyper Text Markup Language yang biasanya digunakan untuk membangun sebuah website. HTML akan membentuk dan menampilkan isi atau tampilan dari sebuah halaman web.

HTML 5 sendiri merupakan versi terbaru dari HTML yang dikembangkan W3C atau World Wide Web Consortium. HTML 5 dikembangkan untuk memperbaiki konten multimedia yang sudah ada pada HTML 4 dan versi sebelumnya agar browser mampu menerjemahkan instruksi-instruksi dengan mudah. HTML 5 merupakan hasil kerja sama antara W3C dengan Web Hypertext Application Technology Working Group atau yang disingkat WHATWG. WHATWG bekerja pada sektor web dan aplikasi sedangkan w#c bekerja pada XHTML 2.0.

Resmi dirilis pada bulan Mei 2011, HTML 5 terus dikebangkan oleh W3C hingga saat ini. Untuk saat ini hampir semua perangkat baik mobile maupun desktop sudah mampu mengimplementasikan HTML 5 dengan cukup baik.

Fitur HTML 5

  • Elemen canvas Tag <canvas> bisa digunakan untuk menggambar secara dinamis dengan bantuan javascript.
  • Elemen VIdeo dan Audio Menggunakan tag <video> untuk menampilkan video dan tag <audio> untuk menampilkan audio.
  • Geolocation Salah satu fitur untuk melacak posisi user dengan menggunakan informasi dari GPS maupun WiFi.
  • Offline Web Application Meskipun tidak terkoneksi dengan internet akan tetapi user masih tetap bisa berinteraksi dengan web.
  • dll

HTML 5 memiliki struktur penulisan yang cukup sederhana dan hanya memiliki dua elemen utama yaitu head dan body. Bagian head diawali dengan tag <head> dan ditutup dengan tah </head>. Sedangkan pada bagian body diawali dengan tag <body> dan ditutup dengan tag </body>. Sebelumnya, pada bagian head dan body diapit dengan tag <html></html>. Pada awal baris penulisan file HTML juga diberikan syntax untuk pendeklarasian yaitu dengan menambahkan tag <!DOCTYPE html>.

Struktur Dasar HTML 5

<!DOCTYPE html>
<html>
  <head>
    <title>Judul</title>
  </head>
  <body>
    <h1>Heading</h1>
  </body>
</html>
